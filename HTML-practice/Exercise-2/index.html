Element

HTML element = start tag+ contant + end tag
Types:
1.Inline Elements: Inline elements do not start on new line and only take as much as width they want.
2.Block Elements:Block elements starts up on new line and pushing any content after them to new line, take up the entire width of their container by default. Width and height can be controlled via CSS

Nested elements
Empty html elements

Attributes:

types:
1.Core Attributes:Basic html elements.
Ex:id,class,style,title.
2.Internationalization attributes:These attributes help to adapt the document to different languages and regions
Ex: lang and dir
3.Generic attributes: These attribute provide additional information about the element.
Ex: data-*
